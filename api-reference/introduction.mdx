Welcome to the **Krumb Developer Docs**
This API reference describes the **RESTful endpoints** you can use to interact with the Krumb URL Shortener platform.  

You can:  
- Shorten links  
- Track analytics  
- Generate QR codes  
- Manage users and API keys  
- Access admin functions (if authorized)  

All endpoints are accessible over **HTTPS** and return **JSON** unless otherwise noted.  
Authentication is required for most operations.
